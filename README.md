    区块分析法为先将现有数据根据一定区块大小进行分块，然后对每个块内数据进行求平均值来代表该块。对每块都进行该操作后这样我们就得到了该区块大小下的一组新数据，最后对新数据进行依次求标准差，标准误差。然后改变区块大小，所有区块大小都计算过后，横坐标为区块的大小，纵坐标为标准误差作图。
    对于划分后有余数的情况，将剩余的数目作为一组。例：若要对20组数据进行每6个划分一次，会分为三组余2个数据，也将这两个数据作为新的一组，即总共有四组。对于多个区块大小可能会划分为相同的组数，即得到的块数相同。对于这种情况则优先选择最平均分组的情况（若有500组数据，每30个分为一组会生成16组余20个数据，共计17组；每31个分为一组时会生成16组余4个数据，共计也17组。每30个一组时每组的数据量最平均，则选择30个一组的划分方式作为会生成17组时的代表，而不是选择31个数据一组）。
